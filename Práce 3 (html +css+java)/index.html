Hrdinové

Hráči vstupují do světa, kde se zrození hrdinové skrývají za zdánlivě obyčejnými tvářemi. Skrze volbu se stávají rytíři či mágové, rozhodujíce o osudu říší. Sbírají zkušenosti, zlepšují dovednosti a taktiku, čelí zkouškám a zdolávají monstra. Přátelství a odvaha formují cestu k hrdinství, zanechávajíc stopu v epičnostech světa fantazie.

Nebezpečí

V propasti stínů se hemží neznámá monstra. Temní démoni se plíží skrze mlhu, drak s plamenným dechem straší nad horizontem. Lesní víly zpívají tajemné písně, zatímco podzemní troglodyti čekají v hladině temných jeskyní. Země se chvěje od kroků obrů, a magická stvoření tančí v lunárním svitu, tvoříc nekonečný bestiář fantazie.

Dobrodružství

V srdci ztraceného lesa, spící hory, či zapomenutých dolů se spočívá mnoho pokladů. Zlaté mince žhnou jako slunce, drahokamy svítí jako zvědavé oči. Mezi artefakty se třpytí zbraň s runami, spoutaná mocí. Vzduch nese vůni dobrodružství. Odvážný nálezce však musí čelit nástahám, neboď né vždy se nachází poklad bez ochrany.
